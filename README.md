# Installation
- [ ] First make a venv
```bash
py -m venv .venv
```
- [ ] Next activate it
```bash
.venv/Scripts/activate
```
or on Mac/Linux
```bash
source .venv/bin/activate
```
- [ ] Install Requirements
```bash 
pip install -r requirements.txt`
```
- [ ] Run the Streamlit
```bash
streamlit run workout_streamlit.py
```
